System Architecture

End-to-End Operational Modules

🛡️

Security & Role Access

Enforced Role-Based Access Control (RBAC) with secure JWT session tokens for Fleet Managers, Safety Officers, Financial Analysts, and Drivers.

🚛

Fleet & Driver Registry

Complete asset lifecycle oversight with real-time status filtering (`Available`, `On Trip`, `In Shop`) and strict driver license expiry validation.

🗺️

Trip State Machine

Automated dispatch state transitions (`Draft → Dispatched → Completed`). Enforces cargo capacity limits and synchronizes vehicle/driver states atomically.

📈

Financial & Fuel Auditing

Live KPI monitoring, ROI calculation per asset, fuel efficiency tracking (km/L), expense categorization, and instant CSV export reports.
